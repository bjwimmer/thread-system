Setup GitHub Sync

To sync your threads across devices, you need a GitHub Personal Access Token. Don't worry - I'll walk you through it!

1. Go to: github.com/settings/tokens/new
2. Note: "Thread Registry Sync"
3. Expiration: "No expiration"
4. Scopes: Check only gist
5. Click "Generate token"
6. Copy the token (starts with ghp_)
