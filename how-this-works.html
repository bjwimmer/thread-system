🧭 The Big Picture
This system is built around one principle: start with the end in mind, then work backward.
🎯 Objectives
The stable anchors of your life. Rarely change. These live on the Long View page and in your North Star.
↓
📌 Goals
Specific outcomes you want to reach in the next 12 weeks. 2–3 per objective. Also on the Long View page, and promoted into the Strategic Life Map when active.
↓
🧵 Threads & Actions
The active work. Each thread is a named pursuit with a single micro-action (5–20 min). These live in the Thread Registry and surface daily on the Morning Map.
📄 What Each Page Does
🗺 Morning Map (index.html)
Your daily starting point. Read your North Star, pick your MVT (Most Valuable Task), set your intent and first move for the session. Scratchpad and Spark Capture for anything that surfaces. Open this first, every day.
⚡ Quick Capture
Brain dump. If a thought, idea, or task surfaces and you don't want to lose it, put it here immediately. Don't organize — just capture. Process it later into threads or goals.
🗺 Strategic Life Map
Your active planning layer. Goals organized by domain across three time horizons — This Week, This Month, and 3 Months. Promote goals forward as they become urgent. Default landing is 3 Months so you always see the full picture first.
🧵 Thread Registry
Every active pursuit lives here as a named thread with a domain, status, notes, and a single micro-action. Two threads are pinned to your weekly Focus Strip on the Morning Map. Threads can be linked to goals on the Strategic Life Map.
🔭 Long View
Your objectives and the goals beneath them. Quiet, out of daily sight, but the source of everything. Use it when doing longer-term thinking, then send goals into the planner with the Make Thread button. Visit periodically — not daily.

⏱ Daily Rhythm
☀️ Morning (5 min)
Open Morning Map. Read North Star. Set MVT. Fill in Intent and First Move. Start the timebox.
🌙 Evening (5 min)
Note one win in the Scratchpad. Update the micro-action on any thread you touched. Capture any loose thoughts in Quick Capture. Hand one thought off to tomorrow.
📅 Weekly (15 min)
Review all active threads in the Registry — reactivate anything that's gone quiet. Update Focus Strip slots for the coming week. Glance at Strategic Life Map to see if anything needs promoting.
☁️ Sync & Data
All data is stored in a single planner-data.json file in a private GitHub Gist. Auto-pull loads your latest data when each page opens. Push is always manual — this prevents one device accidentally overwriting another.

Before switching devices: Push on the device you just used.
If data looks stale: Use Force Pull to override local state.
Security: Your GitHub token is stored only in this browser on this device. Use a fine-grained personal access token scoped to Gists only.
